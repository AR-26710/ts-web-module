# ts-web-module

一个基于TypeScript的Web模块集合，包含多种常用功能组件。

## 项目结构

- `src/modules/`: 包含各个功能模块
  - `bilibili-embed.ts`: Bilibili视频嵌入组件
  - `resource-link.ts`: 资源链接处理组件
  - `text-box.ts`: 文本框组件
- `dist/`: 构建后的输出目录
- `index.html`: 示例页面

## 安装

```bash
npm install
```

## 构建

```bash
npm run build
```

## 使用

1. 引入构建后的JS文件到你的HTML中
2. 根据需要实例化各个模块组件
