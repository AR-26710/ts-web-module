# 画廊组件（带 Shadow DOM）

## 组件介绍

用于创建图片画廊的自定义 Web 组件，支持多张图片展示。

## 使用方式

在 `<gallery-box>` 标签内放置 `<img>` 标签即可，支持任意数量的图片。

## 使用示例

### 画廊展示

<gallery-box>
    <img src="/images/1.jpg" alt="Image 1">
    <img src="/images/2.jpg" alt="Image 2">
</gallery-box>

## HTML 代码

```html
<gallery-box>
    <img src="/images/1.jpg" alt="Image 1">
    <img src="/images/2.jpg" alt="Image 2">
</gallery-box>
```
